# ✨ SoulSync
A student-friendly life RPG built with Streamlit.

## Quick Start
- Add secrets in Space settings:
  - `HF_API_KEY` (optional)
  - `LLM_MODEL_ID` (optional)
  - `DATABASE_URL` (optional)

> Note: Streamlit Spaces run on port 8501 by default. Do not override the port.
